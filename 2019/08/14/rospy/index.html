rospy-Time

  • Time类 反映一个时刻
    指一个时刻，构造一个time对象时，就要指定初始的时间 秒和纳秒，，不填这两个参数的话，就以零时刻开始
    如
    rospy.Time(secs=1,necs=0)#构造函数秒和纳秒
    Time now() 是一个静态的成员函数
    rospy.Time.now()#可以这样调用
  • Duration类 反映一段时间
    rosoy.Duration(1,0) 这个时间有1秒钟，0纳秒
    Time - Time = Duration
    T + D = T
    D + D = D
    D - D = D
    T + T没有意义
