Systick和定时器的使用

Systick介绍

Systick timer 24位向下计数的系统定时器
主要寄存器：
1. STKCTRL(Systick Control and Status register)
2. STKLOAD(Systick reload value register)
3. STKVAL(Systick current value register)
SysTickConfig(SystemCoreClock/1000); //系统时钟设置
SystemCoreClock：72 000 000 1s振荡次数 72MHz
/1000为1ms振荡次数，即将Systick设为了1ms的定时器
中断处理文件stm32f10xit.c中的
void SysTick_Handler(void) { TimingDelay--; //定时器1ms中断一次 }
主函数文件中应用此变量TimingDelay：
void Delay_Ms(u32 nTime) //ms定时器 { TimingDelay = nTime; while(TimingDelay != 0); } 学习要点： 1. Systick介绍 2. 了解SysTickConfig函数 3. 了解void Delay_Ms(u32 nTime)函数原理

介绍定时器（普通、通用）

学习要点：
1. 通用定时器 2. 了解NVIC相关内容
资料包中TIM/TimeBase
内嵌向量中断控制器：Nested Vectored Interrupt Controller(NVIC)
所有ARM3芯片都有的中断向量控制器
统一管理中断向量
/**   * @brief  Configure the nested vectored interrupt controller.   * @param  None   * @retval None   */ void NVIC_Configuration(void) {   NVIC_InitTypeDef NVIC_InitStructure; NVIC初始化结构体   /* Enable the TIM2 global Interrupt */   NVIC_InitStructure.NVIC_IRQChannel = TIM2_IRQn; 中断通道   NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 0; 抢占优先级   NVIC_InitStructure.NVIC_IRQChannelSubPriority = 1; 响应优先级   NVIC_InitStructure.NVIC_IRQChannelCmd = ENABLE; 使能通道中断   NVIC_Init(&NVIC_InitStructure); } 抢占优先级：
A中断正在运行，B有中断可以抢占（打断）A的中断
响应优先级：
- 中断同时来，就执行优先级高的
- 如果B已经中断了，则A就不会抢占它
数字越小 ，优先级就越高
library添加相关的文件
Library/Driver/src
在misc.c文件中（misc混杂的）
/**   * @brief  Configures the priority grouping: pre-emption priority and subpriority.   * @param  NVIC_PriorityGroup: specifies the priority grouping bits length.    *   This parameter can be one of the following values:   *     @arg NVIC_PriorityGroup_0: 0 bits for pre-emption priority   *                                4 bits for subpriority   *     @arg NVIC_PriorityGroup_1: 1 bits for pre-emption priority   *                                3 bits for subpriority   *     @arg NVIC_PriorityGroup_2: 2 bits for pre-emption priority   *                                2 bits for subpriority   *     @arg NVIC_PriorityGroup_3: 3 bits for pre-emption priority   *                                1 bits for subpriority   *     @arg NVIC_PriorityGroup_4: 4 bits for pre-emption priority   *                                0 bits for subpriority   * @retval None   */ void NVIC_PriorityGroupConfig(uint32_t NVIC_PriorityGroup) 优先级组配置 {   /* Check the parameters */   assert_param(IS_NVIC_PRIORITY_GROUP(NVIC_PriorityGroup));      /* Set the PRIGROUP[10:8] bits according to NVIC_PriorityGroup value */   SCB->AIRCR = AIRCR_VECTKEY_MASK | NVIC_PriorityGroup; }
Group0: 0 bits 抢占优先级 4 bits 响应优先级 16个 0-15 高-低
Group1: 1 bits 抢占优先级（2个）可配置为0/1 配置为0的可抢占配置为1的中断 3 bits 响应优先级

定时器配置：
TIM_TimeBaseInitTypeDef  TIM_TimeBaseStructure;  /* Time base configuration */  TIM_TimeBaseStructure.TIM_Period = 1000-1; 计数周期  TIM_TimeBaseStructure.TIM_Prescaler = 72-1; 预分频值 =71 72MHz预分成1M  TIM_TimeBaseStructure.TIM_ClockDivision = 0;时钟分频  TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;计数模式（Up or Down)  TIM_TimeBaseInit(TIM2, &TIM_TimeBaseStructure);

72MHz/72 = 1MHz 即1us 1us * 1000 = 1ms 故设置 TIM_TimeBaseStructure.TIM_Period = 1000-1;

定时器初始化代码：
``` void TIM2Init(void) { TIMTimeBaseInitTypeDef TIM_TimeBaseStructure;
NVICInitTypeDef NVICInitStructure;

    /* Enable the TIM2 global Interrupt */
    NVIC_InitStructure.NVIC_IRQChannel = TIM2_IRQn;
    NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 0;
    NVIC_InitStructure.NVIC_IRQChannelSubPriority = 1;
    NVIC_InitStructure.NVIC_IRQChannelCmd = ENABLE;

    NVIC_Init(&NVIC_InitStructure);

    /* TIM2 clock enable */
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM2, ENABLE);

    /* Time base configuration */
    TIM_TimeBaseStructure.TIM_Period = 1000-1; //1ms
    TIM_TimeBaseStructure.TIM_Prescaler = 71;  //72 000 000 /72 = 1 000 000 Hz = 1MHz
    TIM_TimeBaseStructure.TIM_ClockDivision = 0;
    TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;

    TIM_TimeBaseInit(TIM2, &TIM_TimeBaseStructure);

    /* TIM IT enable */
    TIM_ITConfig(TIM2, TIM_IT_Update, ENABLE); //Update更新，查询中断标志位是否有中断

    /* TIM2 enable counter */
    TIM_Cmd(TIM2, ENABLE);    

}

```

其中，NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 0; 只能设置为0，因为在void NVIC_PriorityGroupConfig(uint32_t NVIC_PriorityGroup) 优先级组配置时，在main函数开头我们将它配置为Group，没有抢占优先级，如果不为0可能有问题
要编写的中断处理函数，有统一的名字，名字在启动文件中给出TIM2中断处理函数名。
例程，stm32f10xit.c 资料包TIM这个文件夹下
/**   * @brief  This function handles TIM2 global interrupt request.   * @param  None   * @retval None   */ void TIM2_IRQHandler(void) {   if (TIM_GetITStatus(TIM2, TIM_IT_Update) != RESET) 判断中断是否到来   {     TIM_ClearITPendingBit(TIM2, TIM_IT_Update);清除中断标志位   } } 注意：中断处理函数，参数设置，TIMIT_Update: TIM update Interrupt source。

端口重映射及蜂鸣器操作方法

端口重映射

蜂鸣器原理图如下：

三极管作用：放大电流
10k电阻作用：限流
二极管作用：与蜂鸣器构成放电回路，防止烧坏三极管
为什么要端口重映射？
因为蜂鸣器的IO口PB4原本的功能是JTAG调试，而非普通IO口
重映射方法：

  1. 开启AFIO复用时钟RCC_APB2Periph_AFIO(所有重映射功能都在AFIO这个时钟管理下面)
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOB|RCC_APB2Periph_AFIO, ENABLE);
  2. 调用GPIO_PinRemapConfig配置管脚重映射
    GPIO_PinRemapConfig(GPIO_Remap_SWJ_NoJTRST, ENABLE);
  3. 注意映射后端口的功能（本实验中重映射后作为普通IO口使用）
    库函数中有例程可以参考GPIO/JTAG_Remap

Stm32时钟简介

  • RCC 复位和时钟总线
  • AHB 系统总线（高阶外设总线）
  1. APB1 定时器、串口、低速
  2. APB2 GPIO、高速
    HSE:高速外部时钟
    HSI:高速内部时钟
    LSI:低速内部时钟
    LSE:低速外部时钟
  3. 了解APB1(低速)和APB2(高速)外设
  4. 学会看STM32时钟框图
  5. 学会用库函数开启外设时钟
  6. 学会STM32 IO配置方法与应用场合

定义初始化结构体

GPIO_InitTypeDef GPIO_InitStructure
定义结构体变量类型 结构体名称
GPIO_Mode
枚举变量构成
GPIO的八种工作模式

  1. GPIO_Mode_AIN
    模拟输入 ADC采样
  2. GPIO_Mode_IPD
    下拉输入 IO口不接的话，默认为低电平
  3. GPIO_Mode_IPU
    上拉输入 IO口不接的话，默认为高电平
  4. GPIO_Mode_IN_FLOATING
    浮空输入 IO口不接的话，状态不定，呈高阻态
  5. GPIO_Mode_Out_OD
    开漏输出，IO输出0接GND，IO输出1，悬空，需要外接上拉电阻，才能实现输出高电平。当输出为1时，IO口的状态由上拉电阻拉高电平，但由于是开漏输出模式，这样IO口也就可以由外部电路改变为低电平或不变。可以读IO输入电平变化，实现C51的IO双向功能，适合于做电流型的驱动,其吸收电流的能力相对强(一般20ma以内)。
  6. GPIO_Mode_Out_PP
    推挽输出，可以输出高电平、低电平（提高电路负载能力，开关速度）
  7. GPIO_Mode_AF_OD
    复用开漏输出，片内外设功能
  8. GPIO_Mode_AF_PP
    复用推挽输出，片内外设功能

IO口配置步骤

  1. 定义一个IO口初始化结构体 GPIO_InitTypeDef GPIO_InitStructure;

  2. 使能相应的IO口时钟 RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC, ENABLE);

  3. 结构体赋值

  4. 调用初始化函数 GPIO_Init(GPIOC, &GPIO_InitStructure);

LED锁存器控制

  • D0-D7对应的GPIO引脚为PC8-PC15
  • LE锁存器控制引脚为PD2

LED为共阳模式

  • GPIO_ResetBits(GPIOC,led_ctrl<<8); 把对应引脚清0，不影响其他位
    例如，将led_ctrl赋值为0x01，则GPIOC的第八个引脚被赋值为0。
  • GPIO_SetBits(GPIOD,GPIO_Pin_2); 把对应的引脚置1，不影响其他位
  • 编写LED控制代码的方法
    例如，使LED亮，设置该LED对应的GPIO口的状态，再打开锁存器使PC口的状态做出的改变输出到LED口。之后，关闭锁存器使该状态保持，并且保持的状态不会改变。
    GPIO_ResetBits(GPIOC, led_ctrl<<8);
  GPIO_SetBits(GPIOD, GPIO_Pin_2);
  GPIO_ResetBits(GPIOD, GPIO_Pin_2);
