#GitHub创建个人仓库

登录GitHub，点击GitHub中的New repository创建新仓库，仓库名为GitHub账号名称.github.io,为固定写法。

#安装Git

Git官网下载：Git-Downloading Package
安装成功后，将你的Git与GitHub账号绑定，鼠标右击打开Git Bash,设置user.name和user.email配置信息：
git config –global user.name “你的GitHub用户名”
git config –global user.email “你的GitHub注册邮箱”
